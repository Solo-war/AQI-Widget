# AQI Widget

Небольшое настольное приложение на Electron, которое показывает качество воздуха (AQI) и концентрацию PM2.5/PM10 для выбранной станции сервиса WAQI.

## Возможности
- Текущий AQI, PM2.5 и PM10 (если доступны).
- Адрес станции и метка времени последнего обновления.
- Автообновление каждые 10 минут и кнопка «Обновить».
- Автозапуск при входе в Windows.
- Глобальная горячая клавиша для показа окна: Ctrl+Alt+Q.
- Цвет верхней панели меняется по официальной шкале AQI.
- Иконка приложения меняется в зависимости от уровня AQI.

## Установка и запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте файл `.env` в корне проекта на основе `.env.example` и задайте токен:
   ```env
   WAQI_TOKEN=ваш_токен_waqi
   ```
3. Запуск приложения в режиме разработки:
   ```bash
   npm start
   ```

## Сборка
Собрать установочный пакет:
```bash
npm run build
```

## Конфигурация
- Переменные окружения читаются через `dotenv` в `main.js`.
- Необходима переменная `WAQI_TOKEN` — токен API WAQI.
- Станция задаётся константой `STATION` в `script.js`.

## Источник данных
Приложение использует API [World Air Quality Index (WAQI)](https://aqicn.org/). Данные предоставляются «как есть», возможны задержки и ограничения по лимитам API. Ответственность за использование данных лежит на пользователе.
